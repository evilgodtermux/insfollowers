## How to use

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/evilgodtermux/insfollowers`

`$ cd insfollowers`

`$ chmod +x insfollowers.sh`

`$ termux-wake-lock`

`$ bash insfollowers.sh`
